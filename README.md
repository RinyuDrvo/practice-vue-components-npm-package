# practice-vue-components-npm-package

このプロジェクトは Vue コンポーネントの npm パッケージ化学習用です

## Project setup

```
npm install
```

## build bundle

```
npm run build-bundle
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
